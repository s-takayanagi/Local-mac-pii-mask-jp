# PII Masker — ドキュメント

このフォルダには PII Masker の設計ドキュメントが収録されています。

## フォルダ構成

```
docs/
├── 0_概要設計/          ← 最初に策定した構想・要件・設計方針
├── 1_アーキテクチャ設計/ ← コンポーネント構成・データモデル
├── 2_詳細設計/          ← パイプライン・ファイルハンドラー・インターフェース
└── 3_設定・開発/         ← 設定値・デプロイ・開発ガイド
```

## 読む順序

| 順 | フォルダ | 目的 |
|----|---------|------|
| 1 | [0_概要設計](0_概要設計/README.md) | プロジェクトの背景・目的・要件を把握する |
| 2 | [1_アーキテクチャ設計](1_アーキテクチャ設計/README.md) | システム全体像とデータ構造を理解する |
| 3 | [2_詳細設計](2_詳細設計/README.md) | 各コンポーネントの実装詳細を確認する |
| 4 | [3_設定・開発](3_設定・開発/README.md) | ローカル環境構築・デプロイ方法を参照する |
